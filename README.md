# PDF Tools - 在线PDF处理工具

![PDF Tools](https://img.shields.io/badge/Version-1.0.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![Browser](https://img.shields.io/badge/Browser-Compatible-brightgreen)

一个现代化、界面美观的在线PDF处理工具，支持页面管理、合并、替换、旋转和图片导出等功能。

## ✨ 主要特性

### 📁 文件管理
- **拖放上传** - 支持拖放或点击上传PDF文件
- **批量导入** - 支持同时导入多个PDF文件
- **实时预览** - 以缩略图形式预览所有页面
- **页面排序** - 拖拽调整页面顺序

### 🔧 页面操作
- **批量选择** - 支持单选、多选、全选
- **旋转功能** - 左旋/右旋90°（支持批量操作）
- **删除页面** - 删除单个或选中的多个页面
- **页面替换** - 用新PDF页面替换选中的页面
- **图片导出** - 将页面导出为PNG图片（支持低、中、高质量）

### 📄 PDF处理
- **合并功能** - 将所有页面合并为一个PDF文件
- **选择性下载** - 可选择下载全部或仅选中的页面
- **批量替换** - 批量替换多个页面为同一PDF的第一页

## 🎯 使用指南

### 1. 导入PDF
- **拖放上传**：直接将PDF文件拖到左侧上传区域
- **文件选择**：点击"选择文件"按钮选择PDF文件
- **支持多个**：可同时导入多个PDF文件

### 2. 页面管理
- **选择页面**：点击页面卡片选择，再次点击取消选择
- **批量操作**：使用左侧控制面板的批量操作按钮
- **拖拽排序**：拖拽页面卡片调整顺序

### 3. 替换功能
1. 点击"启用替换"进入替换模式
2. 选择要替换的页面
3. 点击"选择文件"上传替换用的PDF
4. 确认后完成替换

### 4. 图片导出
1. 点击页面卡片上的"转PNG"按钮
2. 选择图片质量（低/中/高）
3. 确认导出，自动下载

### 5. 下载PDF
- **合并所有**：点击底部"下载"按钮
- **选择下载**：先选择页面，再点击"下载选中"
- **清除内容**：点击底部"清空"按钮

## 📊 性能优化

### 渲染优化
- **懒渲染**：只渲染可见区域的缩略图
- **图片缓存**：已渲染的页面缓存数据URL
- **批量处理**：大量操作分批处理避免阻塞

### 内存优化
- **及时清理**：移除页面时清理相关资源
- **Blob清理**：下载后及时释放Blob URL
- **Canvas重用**：尽可能重用Canvas元素

## 🐛 常见问题

### Q: 文件大小有限制吗？
**A**: 由于在浏览器中处理，建议单个PDF文件不超过100MB，总页数不超过500页。

### Q: 支持哪些PDF格式？
**A**: 支持标准的PDF 1.4及以上版本，支持加密PDF。

### Q: 处理速度慢怎么办？
**A**: 可以尝试减少同时处理的文件数量，或使用更高质量的图片设置。

### Q: 页面丢失或显示异常？
**A**: 刷新页面重新导入，或检查PDF文件是否损坏。
